# OpenAuth

## POST 发送邮箱验证码

POST /api/open/email-code/send

> Body 请求参数

```json
{
  "email": "open_demo_user@example.com",
  "scene": "register"
}
```

### 请求参数

|名称|位置|类型|必选|说明|
|---|---|---|---|---|
|body|body|object| 是 |none|
|» email|body|string| 是 |none|
|» scene|body|string| 是 |login或register|

> 返回示例

> 200 Response

```json
{}
```

### 返回结果

|状态码|状态码含义|说明|数据模型|
|---|---|---|---|
|200|[OK](https://tools.ietf.org/html/rfc7231#section-6.3.1)|none|Inline|

## POST 开放模式登录换 Token

POST /api/open/token

> Body 请求参数

```json
{
  "account": "string",
  "password": "string"
}
```

### 请求参数

|名称|位置|类型|必选|说明|
|---|---|---|---|---|
|body|body|[OpenTokenRequest](#schemaopentokenrequest)| 是 |none|

> 返回示例

> 200 Response

```json
{
  "code": 0,
  "msg": "string",
  "uuid": "string",
  "data": {
    "accessToken": "string",
    "tokenType": "string",
    "expiresIn": 0,
    "refreshToken": "string",
    "refreshExpiresIn": 0,
    "scope": "string",
    "sessionId": "string",
    "user": {
      "userId": "348e4956c7bd4f763b",
      "account": "string",
      "displayName": "string",
      "email": "string",
      "mobile": "string"
    }
  }
}
```

|---|---|---|---|

|401|[Unauthorized](https://tools.ietf.org/html/rfc7235#section-3.1)|开放登录鉴权失败|None|
|403|[Forbidden](https://tools.ietf.org/html/rfc7231#section-6.5.3)|账号锁定或停用|None|
|500|[Internal Server Error](https://tools.ietf.org/html/rfc7231#section-6.6.1)|签发失败|None|
